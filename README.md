# htaccess-geni 

A node module to generate .htaccess files through a simple builder pattern. 

## Installation 

### npm 

```sh 
	npm install htaccess-geni --save-dev
```

### clone git repository 

```sh
	git clone https://github.com/suhdev/htaccess-geni.git 
```
